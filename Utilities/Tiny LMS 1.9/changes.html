Changes

Version 1.8.2
2009-09-30

  • Quiz mode did not work with IE.
  • Moved quiz mode out of TOC into tree actions.
  • Removed "expand all" and "collapse all" tree actions.

Version 1.8
2009-09-09

  • Added support for "Quiz" structure.

Version 1.7.1
2008-12-18

  • TinyLMS 1.7 did not work on a case-sensitive web-server.

Version 1.7
2008-12-16

  • Rewrote Java-Script code.
  • Updated to latest revision of xhtmlrenderer.
  • Fixed wrong encoded base URL's when printing a course as a PDF-document, which caused xhtmlrenderer to miss CSS files.

Version 1.6.3
2008-03-20

  • The "Next" and "Previous" buttons can now be hidden.

Version 1.6.2
2008-03-18

  • Browser support: Fixed rendering of table of contents on IE 6.

Version 1.6.1
2008-01-24

  • PDF Export: SCORM tutorials can now be printed as PDF documents.
  • imsmanifest.xml parsing: TinyLMS requires now that XML-Id's are well formed.

Version 1.5.1
2006-10-13

  • imsmanifest.xml parsing: Continued work on XML namespaces aware parser.

Version 1.5
2006-10-12

  • imsmanifest.xml parsing: If the "adlcp:scormtype" of a "resource" element is missing, TinyLMS will issue a warning and assume "sco", instead of refusing to open a SCORM content package.
  • imsmanifest.xml parsing: Most parsing is done now with an XML namespaces aware parser.

Version 1.4.9
2006-09-06

  • Removed scrollbar from TOC when running with MS IE, due to issues with CSS box model support in this browser.

Version 1.4.8
2006-07-29

  • Fixed Dutch translations.
  • Worked around loss of CMI data when opening a SCO from the index page.
  • The title and buttons in the tree TOC do not scroll away anymore.

Version 1.4.7
2006-07-11

  • Fixed Slovenian and Dutch translations.
  • The tree TOC displays now navigation buttons if the navigation bar is switched off.

Version 1.4.6
2006-06-26

  • Fixed Slovenian translation.
  • Translated course builder interface into Dutch.

Version 1.4.5
2006-06-13

  • Translated runtime interface into Dutch.
  • Translated course builder interface into Slovenian and finished translation of runtime interface.

Version 1.4.4
2006-06-01

  • TOC was sometimes empty after login.
  • Unicode text in TOC needs to be encoded with HTML entities.

Version 1.4.3
2006-05-26

  • Translated course builder interface into German.

Version 1.4.2
2006-05-25

  • TinyLMS course builder only worked on Mac OS X.
  • Fixed issues with Internet Explorer.

Version 1.4.1
2005-06-24

  • Translated runtime interface into Slowenian language (not finished yet).
  • Moved "Previous" and "Next" buttons into navigation bar to the bottom right.
  • Fixed issues with Apple Safari browser.

Version 1.4
2005-06-08

  • TinyLMS can be run as a command line tool.
  • Updated to Quaqua Look and Feel 3.2.1.

Version 1.3.2
2004-06-20

  • TinyLMS did not work for courses consisting of a single SCO only.
  • Log window uses now a stylesheet.
  • Expanding and collapsing of nodes in the tree TOC did not work on some browsers.
  • Hardcoded path to course directory replaced by an instance variable in file lmsapi.js.

 

Version 1.3.1
2004-06-14

  • Logout from a tutorial works now.
  • When the 'menu' page is shown, no item in the table of contents is selected anymore.

 

Version 1.3
2004-06-14

  • Debugging: Log messages are now written into a log window. No more alerts!
    Many thanks to Kent Ogletree for supplying the code for this.
  • Generated courses now work inside parent framesets.
  • Generated courses that concist of a single SCO had illegal framesets.
  • TinyLMS computes the cmi.core.total_time now correctly.
  • Persistency bug introduced with version 1.2.2 fixed.

 

Version 1.2.2
2004-04-16

  • SCORM-to-SCORM adapter feature added.
  • Option "Always display Table of Contents" added.
  • Links from the statistics view did not work.

 

Version 1.1.8

  • This version fixes a bug which caused the navigational elements of TinyLMS to freeze, when an SCO did not call LMSFinish when it was removed from the view.

 

Version 1.1.7

    (prior versions supported German only).
  • The course builder provides now much more options for customizing generated tutorials.
  • Improved the validation of imported learning content.
  • Lots of bugs fixed.
